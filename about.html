About This Project

The Daily Planner / Task Viewer is a web application created using HTML, CSS, and JavaScript.

It helps users organize daily tasks, mark tasks as completed, and track progress in a simple way.

Features

  • Add tasks with name, time, priority, and status
  • Display tasks in a structured table
  • Mark tasks as completed or pending
  • Calculate completed and pending tasks
  • Show completion percentage
  • Use a responsive layout
