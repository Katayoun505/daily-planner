# Daily Planner

A simple, responsive daily planner web app built with HTML, CSS, and JavaScript.  
It helps users organize daily tasks in a clean and easy-to-use interface.

## Features

- Add daily tasks
- Mark tasks as completed
- Clean and responsive layout
- Simple and beginner-friendly code structure
- Works in modern browsers
